WAVingS

This is a game that you have to avoid falling down on the ocean.
The more distance you walk through, the more score you get.
Beware: the game will get harder and harder!!!

Control:
W, K, ↑ : moving up
S, J, ↓ : moving down
A, H, ← : moving left
D, L, → : moving right
